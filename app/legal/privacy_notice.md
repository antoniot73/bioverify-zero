# Aviso de privacidad simplificado

BioVerify-Zero procesa temporalmente dos imágenes proporcionadas por la persona usuaria: una imagen facial proveniente de un documento y una muestra facial capturada mediante cámara.

## Finalidad

La finalidad única es realizar una comparación facial 1:1 durante la sesión actual.

## No almacenamiento deliberado

El sistema no almacena deliberadamente imágenes, documentos, embeddings, plantillas biométricas ni historial de verificaciones.

## Limitación técnica

El prototipo no promete destrucción criptográfica absoluta de memoria, porque el runtime, el servidor, NumPy, OpenCV y el sistema operativo pueden crear copias temporales internas.

## Uso de datos

Las imágenes no se usan para entrenar modelos, crear perfiles, identificar personas contra bases de datos ni inferir atributos sensibles.

## Resultado

El resultado es probabilístico y no constituye validación legal de identidad.
